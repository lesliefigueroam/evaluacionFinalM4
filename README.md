# Administrador de Usuarios - API REST

Aplicación web que consume el servicio REST de [JSONPlaceholder](https://jsonplaceholder.typicode.com/users)
y administra la información de los usuarios mediante una clase de JavaScript.

## Objetivos de la aplicación

- Consumir un web service externo usando `XMLHttpRequest` desde el constructor de una clase.
- Almacenar la data dentro de la clase para reutilizarla sin nuevas peticiones.
- Exponer métodos que permiten:
  - Listar los nombres de todos los usuarios.
  - Buscar un usuario por nombre (prompt) y mostrar su información básica (username y correo).
  - Buscar un usuario por nombre (prompt) y listar su dirección completa.
  - Buscar un usuario por nombre (prompt) y listar su información avanzada (teléfono, sitio web y compañía).
  - Listar todas las compañías junto a su frase clave (catchphrase).
  - Listar los nombres de los usuarios ordenados alfabéticamente.
- Disponer de un botón HTML para invocar cada método.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Programación Orientada a Objetos con clases, XMLHttpRequest)

## Cómo ejecutar

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. Presiona los botones para ejecutar cada método.
4. Abre la consola del navegador (tecla F12 → pestaña "Console") para ver las salidas detalladas.
